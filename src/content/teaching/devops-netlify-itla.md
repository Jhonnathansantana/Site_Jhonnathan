Taller orientado a estudiantes y profesionales que desean automatizar sus flujos de despliegue y adoptar prácticas DevOps en proyectos reales.

## Contenidos principales

- Control de versiones con Git y ramas protegidas.
- Integración continua con GitHub Actions.
- Despliegue continuo en Netlify.
- Variables de entorno y gestión de secretos.
- Funciones serverless para extender sitios estáticos.
- Buenas prácticas de monitoreo y rollback.
